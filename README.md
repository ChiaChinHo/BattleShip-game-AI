# BattleShip-game-AI

# Command
python BattleShipWithGui.py [plyer1] [player2] -g -s -n [1]＜/br＞ 

player1, player2 : player type, [s1, s2, mc, dp, rd], default: rd＜/br＞ 
    s1 : Strategy 1＜/br＞ 
    s2 : Strategy 2＜/br＞ 
    mc : Monte Carlo＜/br＞ 
    dp : dynamic programming＜/br＞ 
    rd : random＜/br＞ 

-g : GUI＜/br＞ 
-s : stop every step＜/br＞ 
-n : number of games＜/br＞
